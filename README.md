# carrinho-de-compras-js

Um projeto simples de front-end que simula um carrinho de compras.
O usuário pode adicionar produtos, definir quantidades e visualizar o valor total automaticamente.

Tecnologias usadas

* HTML5
* CSS3
* JavaScript

Funcionalidades

* Adicionar produtos ao carrinho
* Cálculo automático do valor total
* Atualização dinâmica da interface
* Limpar carrinho

Como funciona

* O usuário seleciona um produto e define a quantidade
* O sistema calcula o valor com base no preço unitário
* O item é adicionado ao carrinho
* O total é atualizado automaticamente

Aprendizados

* Manipulação do DOM
* Operações matemáticas com JavaScript
* Uso de variáveis globais
* Atualização dinâmica de elementos

Possíveis melhorias

* Evitar duplicação de produtos no carrinho
* Adicionar botão de remover item
* Salvar carrinho no localStorage
* Melhorar validação (ex: quantidade inválida)
